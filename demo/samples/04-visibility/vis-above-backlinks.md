# Teste de visibilidade — above-backlinks

**Cenarios:**
1. Backlinks core plugin ON + Backlinks in document ON → callout acima dos backlinks (DOM)
2. Backlinks in document OFF → fallback pro CM6 bottom
3. Backlinks core plugin OFF → fallback pro CM6 bottom
4. Religar tudo → callout volta pra posicao DOM (reatividade)
