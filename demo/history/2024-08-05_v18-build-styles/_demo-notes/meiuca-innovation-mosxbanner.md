Me ajude a construir um plugin para Obsidian.md, usando node.js, API do obsidian e se necessário, algum facilitador como svelte ou react.

O objetivo deste plugin é criar uma ferramenta para pesquisadores de dados qualitativos que querem codificar seus dados.&nbsp;

A utilização proposta se dá da seguinte maneir<span class="coded-text" style="background-color:#FFFF00;">a: o usuário im</span>porta a transcrição completa de uma <span class="coded-text" style="background-color:#FFFF00;">entrevista no Obsidian</span>. Ao percorrer e analisar o documento, ele seleciona trechos de texto com o mouse e atribui um código a cada seleção. Após codificar o conteúdo, o usuário pode comparar e analisar esses códigos, formando temas. Neste momento,<span class="coded-text" style="background-color:#FFFF00;"> nosso foco é o processo de codificação dos trechos</span> de texto. Imaginei que cada trecho selecionado será destacado com uma cor de fundo correspondente ao código atribuído. <span class="coded-text" style="background-color:#FFFF00;">Assim, ao selecionar um trecho de texto, aparecerá automaticamente uma opção ao lado para a</span>dicionar um novo código ou escolher entre os já existentes, sem a necessidade de usar o botão direito do mouse. A<span class="coded-text" style="background-color:#FFFF00;"> estrutura geral e os templates</span> de plugins já estão instalados no Node, prontos para serem programados. Agora, <span class="coded-text" style="background-color:#FFFF00;">podemos prosseguir </span>diretamente com a codificação.
